# mabongpapa-tistory-skins
- 티스토리 스킨을 만들기 위한 프로젝트입니다.

## 할일
- express 서버 만들기 (대상 dist)
- html to ejs 만들기
- html to 테터툴즈 만들기
- sass 컴파일 만들기
- ts to js 만들기 (번들 해야 하나?)


## 사용법

1. **의존성 설치**

   프로젝트에 필요한 패키지들을 설치합니다.

   ```bash
   npm i gulp-cli -g
   npm install
   ```

2. **개발 모드 실행**

   아래 명령어를 실행하면 소스 파일(`src`)의 변경사항을 감지하여 자동으로 빌드한 후, 결과물을 `dist` 폴더에 생성합니다.

   ```bash
   npm run dev
   ```

3. **결과물 확인**

   `dist` 폴더에 생성된 스킨 파일들을 확인하고 티스토리에 적용할 수 있습니다.
